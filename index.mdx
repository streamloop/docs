Welcome to the official documentation for **Streamloop** ([streamloop.app](https://streamloop.app))
— the cloud tool that loops your pre-recorded videos and audio into a continuous **24/7 live
stream** on YouTube or any RTMP platform. No OBS, no encoding hardware, no PC left running.

New here? The [quickstart](/quickstart) takes you from a video file to a live stream in a few
minutes.
